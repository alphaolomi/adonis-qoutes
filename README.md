# Qoute

A simple Qoutes web app built in AdonisJs, it comes pre-configured with.

## Features

1. Bodyparser
2. Authentication
3. Web security middleware
4. CORS
5. Seeds qoutes

## Setup

Use the Yarn command to install

```bash
yarn
```

### Migrations

Run the following command to run startup migrations.

```js
adonis migration:run
```
